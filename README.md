# yPrompter

Schedule agent prompts for later.

yPrompter is a tiny macOS and Windows tray app that lets users schedule a Codex CLI prompt to run later against a selected local repository.

## Requirements

- macOS or Windows
- Codex CLI installed
- User already logged into Codex CLI
- Computer awake when scheduled jobs run
- yPrompter running in the tray/menu bar

## v0.1 Scope

- One scheduled prompt at a time
- Repository folder picker
- Prompt editor
- Run time picker
- Plan / Goal / Execute modes
- Sandbox selection
- Approval selection
- Run Now
- Schedule
- Cancel Scheduled Run
- Open Last Log

## Not in v0.1

- Payments
- Accounts
- Cloud sync
- Six-prompt queue
- Usage pacing graph
- Native Windows Task Scheduler integration
- Native macOS launchd integration
- Code signing or notarization

## Distribution

v0.1 uses unsigned beta builds. macOS and Windows may show security warnings.
